初中数学：二次函数综合题

"已知抛物线 y = ax² + bx + c 经过点 A(-1,0) 和 B(3,0)，且与 y 轴交于点 C(0,3)。求抛物线的解析式，并判断点 D(2,4) 是否在该抛物线上..."

难度：中等偏难
知识点：解析几何

account_tree 逻辑链路图

1

确定解析式形式

根据交点 A, B 选择两点式方程 y = a(x+1)(x-3)

check_circle
2

代入点 C 求系数

代入 (0,3) 得到 3 = a(0+1)(0-3) → a = -1

check_circle
3

展开解析式 (逻辑偏移点)

错误展开为 y = -x² - 2x + 3 (应为 -x² + 2x + 3)

运算错误
error
tips_and_updates

逻辑修正建议

在第3步进行括号展开时，负号分配错误。建议在处理系数 a = -1 时，先展开括号内部，再整体乘以 -1。

正确路径：y = -(x² - 2x - 3) = -x² + 2x + 3

大白话版

解题妙招：想象括号前面那个负号是一个"收纳袋"，你在整理里面的包裹（多项式）时，要把每一个小快递都贴上"反转"的标签。你这次只给第一件衣服贴了标签，后面的小件都漏掉了哦！

shield 关键避坑指南

  • • 二次函数解析式展开时，系数 a 必须在最后一步再分配进括号，防止多项式符号混乱。
  • • 两点式 y = a(x-x1)(x-x2) 的中间项符号是由 -(x1+x2)x 决定的，这里要注意 -1 和 3 的和是 2，变号后是 -2，但前面还有个 a。
